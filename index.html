It's Synchronous:

ScriptSmith is a collaborative web-based code editor that allows multiple developers to work together in real-time on the same project. It provides a seamless coding experience with features like syntax highlighting, auto-completion, and error highlighting. Our platform aims to make coding easier and more efficient for developers by eliminating the need for constant file sharing and merging. With ScriptSmith, developers can focus on writing great code while our platform takes care of the collaboration and version control. Join us and experience a new way of coding!
